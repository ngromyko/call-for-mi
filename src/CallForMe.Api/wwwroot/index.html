OpenAI

Ключ не показывается после сохранения. Если ошиблись, просто вставьте новый и сохраните ещё раз.

Twilio

PublicBaseUrl должен быть публичным HTTPS URL, доступным Twilio. Для локальной разработки обычно нужен tunnel: ngrok или Cloudflare Tunnel.
